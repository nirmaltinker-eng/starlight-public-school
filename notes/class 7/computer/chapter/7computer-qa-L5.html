3. निम्नलिखित प्रश्नों के उत्तर लिखिए (Question-Answer)

1. प्रश्न: 'C' भाषा से आप क्या समझते हैं?

उत्तर: 'C' एक **उच्च-स्तरीय (High-Level) प्रोग्रामिंग भाषा** है। इसका विकास **डेनिस रिची (Dennis Ritchie)** ने **1972** में **बेल लेबोरेटरीज (Bell Laboratories)** में किया था। यह अपनी दक्षता (Efficiency) और प्रोग्रामिंग में व्यापक उपयोग के कारण बहुत लोकप्रिय है। इसे पुराने भाषाओं जैसे- FORTRAN, ALGOL, PL/I की तुलना में अधिक महत्त्व दिया गया था।

2. प्रश्न: 'C' भाषा में प्रयोग होने वाले की वर्ड्स के नाम बताइए।

उत्तर: 'C' भाषा में **32 की वर्ड्स (Keywords)** प्रयोग होते हैं, जिनमें से कुछ निम्नलिखित हैं:
auto, break, case, const, continue, default, do, else, enum, extern, float, for, goto, if, int, long, register, return, short, signed, sizeof, static, struct, switch, typedef, union, unsigned, void, volatile, while.

3. प्रश्न: फ्लोचार्ट के चिन्ह समझाइए।

उत्तर: फ्लोचार्ट (Flowchart) किसी प्रोग्राम की तार्किक प्रक्रिया को ग्राफ़िक रूप से दर्शाने के लिए इस्तेमाल किए जाने वाले आरेख (Diagram) होते हैं। इसमें विभिन्न कार्यों को दर्शाने के लिए विशिष्ट चिन्हों (Symbols) का उपयोग किया जाता है:

आरंभ/समाप्ति (Start/End): अंडाकार (Oval) $\to$ प्रोग्राम शुरू या समाप्त करना।
इनपुट/आउटपुट (I/O): समानांतर चतुर्भुज (Parallelogram) $\to$ डेटा पढ़ना या परिणाम प्रिंट करना।
प्रक्रिया/संसाधन (Process): आयत (Rectangle) $\to$ गणनाएँ (Calculations) या डेटा प्रोसेसिंग।
निर्णय (Decision): हीरा/समचतुर्भुज (Diamond) $\to$ शर्त (Condition) की जाँच करना (हाँ/नहीं, सत्य/असत्य)।
प्रवाह रेखाएँ (Flow Lines): तीर (Arrows) $\to$ निर्देशों के क्रम (Flow of Control) को दर्शाना।

4. प्रश्न: इंटरचेंज (Swapping) का प्रोग्राम किस प्रकार बनेगा?

उत्तर: इंटरचेंज (Swapping) प्रोग्राम दो वेरिएबल के मानों को आपस में बदलने के लिए एक **तीसरे अस्थायी वेरिएबल (temp)** का उपयोग करता है।
C भाषा में इसका प्रोग्राम (Swap two numbers using a temporary variable) इस प्रकार बनेगा:

#include <stdio.h>
int main() {
    int a = 10;
    int b = 20;
    int temp;

    printf("स्वैप से पहले: a = %d, b = %d\n", a, b);

    // इंटरचेंज (Swapping) लॉजिक
    temp = a; // a का मान temp में रखें
    a = b; // b का मान a में रखें
    b = temp; // temp (पुराना a) का मान b में रखें

    printf("स्वैप के बाद: a = %d, b = %d\n", a, b);
    return 0;
}

5. प्रश्न: लीप ईयर (Leap Year) के प्रोग्राम को भाषा एवं फ्लो चार्ट से समझाइए।

उत्तर: वह वर्ष जो **4 से पूरी तरह विभाज्य** हो, लेकिन **100 से विभाज्य न हो**, या वह वर्ष जो **400 से विभाज्य** हो, लीप वर्ष कहलाता है।
इसकी प्रोग्रामिंग के लिए सशर्त ब्रांचिंग (IF-ELSE) का उपयोग किया जाता है।
'C' प्रोग्रामिंग कोड:

#include <stdio.h.gt;
int main() {
    int year = 2024;
    // लीप वर्ष की शर्त (Logic for Leap Year)
    if ( (year % 4 == 0 && year % 100 != 0) || year % 400 == 0 ) {
        printf("%d लीप वर्ष है।\n", year);
    } else {
        printf("%d लीप वर्ष नहीं है।\n", year);
    }
    return 0;
}
फ्लोचार्ट (Flowchart) लॉजिक:
•   **आरंभ** $\to$ वर्ष को **इनपुट** करें $\to$ **निर्णय 1:** क्या वर्ष 4 से विभाज्य है? (हाँ/नहीं) $\to$ यदि हाँ, **निर्णय 2:** क्या वर्ष 100 से विभाज्य है? $\to$ यदि नहीं, तो **"लीप वर्ष"** प्रिंट करें $\to$ यदि हाँ (निर्णय 2 में), **निर्णय 3:** क्या वर्ष 400 से विभाज्य है? $\to$ यदि हाँ, तो **"लीप वर्ष"** प्रिंट करें $\to$ अंत में **समाप्ति**।
